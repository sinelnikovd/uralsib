О ЗАКАЗЧИКЕ

Финансовая корпорация «УРАЛСИБ» — это одна из крупнейших российских финансовых групп, которая предоставляет широкий спектр услуг: розничные и корпоративные банковские продукты, инвестиционно-банковские услуги, брокерское обслуживание, управление активами и благосостоянием и другое.

Концепция

Основная задача, стоявшая перед нами, заключалась в создании информационной площадки для финансовой корпорации - общей для всех подразделений компании. Важным было грамотно структурировать большой объем информации.

Клиентам

Мы создали стартовую навигационную страницу для того, чтобы клиенты могли быстро найти необходимую услугу и перейти на нужный сайт.

ИНВЕСТОРАМ

Мы разместили цифры, факты, аналитику, новости для привлечения инвесторов. Посвятили один из главных разделов отчетности и описанию реализованных на инвестициях проектов

ЖУРНАЛИСТАМ

Мы создали образ открытой динамичной компании, разместив отдельным разделом новости, медиаконтент, онлайн-версии печатных корпоративных изданий.

Дизайн

Чтобы не рассеивать внимание пользователя и соответствовать статусу корпоративного ресурса финансовой корпорации, в качестве приоритетов для дизайна были определены строгость, простота графики и поддержка корпоративного брендбука

Легкость элементов интерфейса, яркие однотонные подложки позволяют указать на приоритетную информацию.

Полезные функции

НОВОСТИ ДЛЯ КАЖДОГО

Нами реализовано решение по генерации персональной новостной ленты для каждого пользователя. С помощью кнопки «+ ИЗБРАННОЕ» можно добавить любую новость в собственную ленту. Готовый отчет можно выгрузить в формат doc или pdf

ПРОСМОТР ГОДОВОЙ ОТЧЕТНОСТИ

Для всех пользователей доступны публичные интерактивные отчеты, которые можно просматривать в онлайне без скачивания

КАЛЕНДАРЬ МЕРОПРИЯТИЙ

Посетители всегда могут быть в курсе предстоящих событий Корпорации. На странице выводится список ближайших мероприятий, упорядоченных по дате и типу встречи. Зарегистрированным пользователям доступны напоминания о мероприятиях.
